# Qooxdoo Test Runner

A GUI framework application to run the unit tests

## Developing test runner
Clone this repo and compile it:

```
    $ git clone https://github.com/qooxdoo/qxl.testrunner
    $ cd qxl.testrunner
    $ qx pkg install
    $ qx serve
```
Then open [http://localhost:8080](http://localhost:8080)
